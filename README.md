# Next.js Learning Project

A Next.js project for learning and experimenting with modern React development using TypeScript.

## Tech Stack

- **Next.js** 16.1.0 - React framework for production
- **React** 19.2.3 - JavaScript library for building user interfaces
- **TypeScript** 5.9.3 - Typed superset of JavaScript
- **react-error-boundary** 6.0.0 - Error handling for React components

## Prerequisites

- Node.js (14.17 or higher recommended)
- npm or yarn package manager

## Installation

Clone the repository and install dependencies:

```bash
npm install
```

## Development

To run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
